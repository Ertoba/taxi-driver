# DropTaxi driver apps

Licensed DropTaxi 2.3.0 driver applications for the private `Ertoba/taxi-driver` repository.

- `android/` — Cordova Android driver project
- `ios/` — Cordova iOS driver project

Firebase files, signing keys, provisioning profiles, and real `build.json` files are intentionally excluded from Git. Use the provided `build.example.json` templates and keep production values outside the repository.

The final Android/iOS rebuild is deferred until the backend domain, official license activation, Firebase, Google Maps APIs, notifications, authentication, background location, and Georgian localization are fully configured and verified.
